# Introdução ao Git e GitHub

Um guia web para aprender os conceitos fundamentais de Git e GitHub, desenvolvido especialmente para alunos que se encontram a dar os primeiros passos em programação.

## Sobre o Projeto

Este é um site que apresenta uma introdução ao Git e GitHub de forma clara, visual e prática. O tutorial está organizado em secções navegáveis que guiam o utilizador desde os conceitos básicos até à prática guiada.

## Conteúdo do Tutorial

### 1. **Introdução**
- O que é o Git e o GitHub
- Vantagens de usar controlo de versões
- Relação entre Git (local) e GitHub (remoto)

### 2. **Conceitos Básicos**
- Repositório
- Commit
- Branch (Ramificação)
- Clone, Push e Pull
- Fluxo de trabalho

### 3. **Comandos Git**
Comandos essenciais explicados:
- `git init` - Iniciar repositório
- `git status` - Verificar estado
- `git add` - Adicionar ficheiros
- `git commit` - Guardar alterações
- `git log` - Ver histórico
- `git branch` - Gerir ramificações
- `git push/pull` - Sincronizar com GitHub

### 4. **GitHub**
- Como criar um repositório no GitHub
- Ligar um repositório local ao GitHub
- Clonar repositórios existentes

### 5. **Prática Guiada**
Tutorial passo a passo para:
- Criar primeiro repositório Git
- Fazer primeiro commit
- Enviar projeto para o GitHub

## Como Usar

1. Abra o ficheiro `index.html` num navegador web.
2. Navegue pelas diferentes secções usando os botões no topo.
3. Siga o conteúdo sequencialmente ou aceda à secção desejada.
4. Na secção "Prática Guiada", execute os comandos apresentados.

## Requisitos

- Navegador web
- Git instalado no computador (para praticar os comandos)
- Conta no GitHub (para a atividade prática)
